Insight Matrix


Insight Matrix (IM) is a pattern finding tool for researchers to identify relationships among data set. IM utilizes the theory behind Cluster Analysis as it's programatic approach to group things together and thus benefits the user to better organize the data.
To use IM, you first need to have a dataset in hand either pre-existing or created from scratch. You can either start a new analysis or import your data csv to IM using the 2 buttons on the top right corner.
